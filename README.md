# API_Test_Hostel_Managemet (Postman)

## Overview
This project contains Postman collections for testing APIs of an ERP system (Odoo-based).

## Scope
- Authentication APIs
- Product APIs
- Order APIs

## Tools
- Postman
- Newman (optional)

## How to run
1. Import collection & environment into Postman
2. Select environment
3. Run collection
